Projeto de Vídeos

Projeto construído para treinar a inserção de vídeos em páginas web.

Saiba mais...

Este projeto foi desenvolvido para praticar a implementação de vídeos em sites, explorando diferentes formas de incorporação, como HTML5 <video> e embeds de plataformas externas.

Tecnologias utilizadas: HTML5, CSS3.

Habilidades treinadas:
• Inserção de vídeos locais e hospedados online
• Controles personalizados (play, pause, volume)
• Responsividade e adaptação do player ao layout
• Organização de mídia dentro de componentes visuais

O objetivo foi criar uma experiência simples e eficiente para exibir conteúdo em vídeo, garantindo boa usabilidade em diferentes tamanhos de tela.

Projeto Cordel

Projeto construído para treinar efeito parallax em imagens.

Saiba mais...

Este projeto foi desenvolvido para praticar o efeito parallax, uma técnica visual muito utilizada em sites modernos para criar profundidade e movimento enquanto o usuário rola a página.

Tecnologias utilizadas:
HTML5, CSS3.

Habilidades treinadas:
• Criação de efeitos visuais usando background fijo
• Composição de seções com imagens e textos sobrepostos
• Construção de páginas mais dinâmicas e agradáveis
• Responsividade com elementos em movimento

O objetivo foi replicar o estilo de cordel, combinando tipografia marcante, imagens com profundidade e uma narrativa visual fluida ao longo da página.

Projeto Redes Sociais

Projeto construído para treinar responsividade e múltiplos medias queries e divulgar as suas principais redes sociais.

Saiba mais...

Este projeto foi criado para treinar a construção de páginas totalmente responsivas, utilizando múltiplos media queries para ajustar o layout em diferentes tamanhos de tela — desde celulares até monitores maiores.

O design simula uma página de apresentação de perfis, com links para redes sociais, cards informativos e elementos flexíveis que se reorganizam conforme o espaço disponível.

Tecnologias utilizadas:
HTML5, CSS3 (Flexbox e Media Queries).

Habilidades treinadas:
• Técnica mobile-first
• Criação de layouts fluídos e adaptáveis
• Uso de múltiplos breakpoints
• Organização de cards com ícones e links sociais
• Boas práticas de responsividade

O objetivo foi construir uma página moderna para apresentar suas redes sociais de forma clara, responsiva e visualmente agradável.

Projeto Album

Projeto construído para treinar conteúdos flexíveis usando CSS FlexBox para mostrar um albúm de fotos.

Saiba mais...

Este projeto foi criado para praticar a construção de layouts flexíveis com CSS Flexbox, organizando uma galeria de fotos que se adapta automaticamente ao espaço disponível em diferentes tamanhos de tela.

O layout simula um álbum moderno, no qual as imagens se rearranjam de forma fluida, mantendo espaçamento, alinhamento e proporção mesmo quando o usuário redimensiona a janela ou acessa pelo celular.

Tecnologias utilizadas:
HTML5, CSS3 (Flexbox).

Habilidades treinadas:
• Uso avançado de Flexbox
• Criação de galerias adaptáveis
• Distribuição automática de elementos
• Ajuste dinâmico de imagens em colunas flexíveis
• Organização visual limpa e responsiva

O objetivo foi construir um álbum simples, elegante e totalmente flexível, que apresenta as imagens de maneira harmoniosa em qualquer dispositivo.
